SDN Portfolio Visuals Package
=============================

This package was extracted from the project report starting from Chapter 4 (Implementation), with a small set of dashboard UI screenshots added from the testing/results section because they are visually useful for a portfolio.

Folders
-------
1) 01_curated_core_diagrams
   Best architecture / flow / lifecycle visuals for a polished portfolio case study.

2) 02_curated_dashboard_ui
   Best interface screenshots to show the product/dashboard visually.

3) 03_all_chapter4_extracted
   All extracted Chapter 4 figure images from Figure 34 to Figure 61.

Recommended showcase order
--------------------------
- High-Level System Architecture
- End-to-End Decision and Enforcement Dataflow
- Simplified Campus Topology and Segmentation
- Controller Responsibilities and Module Interactions
- DHCP Binding Lifecycle
- ARP Inspection Decision Flow
- Portal-to-Client Identity Association
- RBAC Policy Evaluation Pipeline
- Enforcement Outcomes
- PAM-to-Controller Authorization Flow
- Proof TTL Timeline and Auto Re-Quarantine
- Session Lifecycle Management
- Dashboard Data Sources and Flow
- Then add 2 to 4 dashboard UI screenshots from folder 02

Note
----
Some figures in the report are terminal/log evidence rather than portfolio-grade visuals. Those were excluded from the curated folders but kept in the all_chapter4 folder when they belong to Chapter 4.
