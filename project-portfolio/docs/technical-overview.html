Architecture

Layered Security Flow

1. Endpoint Discovery

Hosts are tracked with binding context (MAC/IP) at edge switches.

2. RADIUS Admission

Initial NAC decision is enforced before broad service reachability.

3. SDN Policy Control

Role-aware allow/deny decisions are converted to forwarding behavior.

4. Kerberos Step-up

Protected services require ticket-based proof before privileged access.

5. Runtime Enforcement

Sessions, rate-limits, and lockdown controls remain active throughout runtime.

6. Operational Visibility

Dashboard pages expose service health, sessions, events, network, and metrics.
